# AI OLX PC Hunter
An n8n bot for parsing OLX using JS Regex and the Llama 3.1 neural network via Groq

# Project Team
* **Shyngys** @heychinga - n8n logic, Groq Cloud, Telegram bot
* **Miras** @mvkanm - JS parser writing (Regex), HTTP requests, OLX blocking bypass

# Technologies
* **n8n** = the automation core
* **Groq Cloud (Llama 3.1)** = AI-powered profitability analysis and defect detection
* **JavaScript (Regex)** = custom parsing of hidden data from OLX
* **Telegram API** = instant notifications.

# How it works
The system loops through 52 ads, opens each one, extracts the description, and sends data to Groq. The AI ​​evaluates the hardware (weeding out old CPUs and GPUs used for mining) and sends us only the best ones
